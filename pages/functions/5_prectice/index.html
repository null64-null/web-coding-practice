5. 関数・メソッド練習

進捗をみながらカスタマイズ

関数とは

定義のしかた

・function 関数名( 引数 ) { 処理 }

   ・引数は何個あってもいい、なくてもいい

   ・処理は自由になんでも、何行でも書いていい

定義の例１

function aisatu(name){

    const greet = `${name}さん、こんにちは`

    console.log(greet)

}

nameという引数を受け取って、nameに対する挨拶文を作成し、コンソールに表示する関数

定義の例２

function tashizann(a, b){

    const result = a + b

    console.log(result)

}

aとbという引数を受け取って、足し算をして、コンソールに答えを表示する関数
