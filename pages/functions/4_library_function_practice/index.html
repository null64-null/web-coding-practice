4. 用意された関数とメソッド（練習）

htmlやcssに関するメソッド

【参考記事】jQueryの「メソッド一覧」と使い方

HP上の操作などで発火させるメソッド・他のメソッド（一例）

.on("action-name", func)

・第1引数"action-name"は、操作の種類（string型で"click"など）

・第2引数funcは関数（関数の引数として関数を入れる!）

・このメソッドを適応する要素に対し、"action-name"の操作がされた時、funcの関数を実行する

例: ボタンが押されたら「push」とコンソール出力する（コードも参照）

$('.button-class').on( "click", function(){console.log("push")} )

.ready(func)

・ $(document).ready( function(){ ... } ) のこと

・jQueryを書く時、メインコードを{ ... }の中に書いているあれ

・実はこれも関数（メソッド）で、"ページ全体"$(document)に対し、ページ全体が表示された後、引数の関数のを実行する。という意味になっている
