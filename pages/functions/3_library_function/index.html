3. 用意された関数とメソッド（解説）

$() について

・関数らしい。jQuery()ともかけるそうだ

・jQueryにもともと備わっている、特殊な関数

・引数にクラス名やタグ名を渡すと、そのクラス名やタグ名を持つ要素自体を返り値として返す

$('.class-name').addClass('new-class-name') のような書き方

・addClass()は「メソッド」と呼ばれるもの

・取得した要素の後に「.」をつけて繋げて使う

・取得した要素に対して何かしらの処理を行う（例では、要素にnew-css-classという新しいCSSクラスをつけている）

・addClass('new-class-name')の'new-class-name'は引数であり、関数と全く同じ使い方をする（ちなみにaddClassの場合は返り値は返していみたい）

・つまり、メソッドは、関数の一種で「ドットで繋げて使う関数」

・メソッドは、jQuery, JavaScriptで元々のたくさん用意されている（自分で作ることはあまりなさそう）

・つまり、メソッドは概ね「すでに用意されている、ドットで繋げて使う関数」
